# lfexpctnchk

余命チェック by Python

## 概要
誕生年月日と希望寿命年数を入れると、
希望寿命までの総日数と本コマンド実行日からの残日数を出力します。

## Usage
<pre>
lfexpctnchk [-v]
      希望寿命までの日数を算出します
      -v : 年毎の月日数表示を行います
</pre>
